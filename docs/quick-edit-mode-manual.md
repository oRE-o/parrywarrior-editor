# 빠른 편집 모드 매뉴얼

## 목표

빠른 편집 모드는 새 에디터에서 **노래를 재생하면서도** 키 입력으로 노트를 배치할 수 있게 해주는 편집 방식입니다.

핵심은 실시간 녹음처럼 생(raw) 시간에 박는 것이 아니라, **기존 타임라인 클릭 배치와 같은 스냅 규칙과 중복 방지 규칙을 그대로 재사용**한다는 점입니다.

이 모드에서 기대하는 동작은 다음과 같습니다.

- 재생 중에도 편집 가능
- 레인 키 입력으로 현재 시점의 스냅된 위치에 노트 생성
- 동시 입력으로 여러 레인에 동시치기 생성
- 롱노트 가능 타입은 키 다운 시작 / 키 업 종료
- 빠른 편집 모드에서는 마우스 클릭으로 노트 삽입 불가

## 1. 빠른 편집 모드 켜기 / 끄기

빠른 편집 모드는 아래 방식으로 전환합니다.

- 전용 빠른 편집 토글 키
- 에디터 UI의 빠른 편집 토글 버튼

`Q`는 빠른 편집 모드 토글에 사용됩니다.

## 2. 기본 동작

빠른 편집 모드가 켜져 있고 곡이 재생 중일 때:

1. 레인 키를 누르면 해당 레인에 노트를 배치한 것처럼 처리됩니다.
2. 배치 시점은 현재 재생 위치를 기준으로 하지만, 최종 배치는 항상 **스냅 적용 후 시간**을 사용합니다.
3. 기존 클릭 배치와 동일하게 **중복 / 충돌 방지 규칙**을 적용합니다.
4. 현재 선택된 노트 타입이 탭 노트면 즉시 노트를 생성합니다.
5. 현재 선택된 노트 타입이 롱노트 가능 타입이면 키 입력 시작과 종료를 사용해 롱노트를 만듭니다.

즉, 빠른 편집 모드는 별도의 우회 저장 경로가 아니라 **기존 클릭 배치 로직을 재사용하는 입력 모드**여야 합니다.

## 3. 기본 키 배치

기본 키 배치는 레인 수에 따라 달라집니다.

### 3레인

- `D F J`

### 4레인

- `D F J K`

### 5레인

- `S D F/J K L`
- 가운데 레인은 `F`와 `J`를 공유합니다.

### 6레인

- `S D F J K L`

### 7레인

- `S D F Space J K L`

## 4. 키 설정 변경

빠른 편집 모드의 키 배치는 설정 팝업에서 바꿀 수 있습니다.

예상 흐름은 다음과 같습니다.

1. 빠른 편집 키 설정 팝업을 엽니다.
2. 변경할 레인 항목을 클릭합니다.
3. 원하는 키를 누릅니다.
4. 저장합니다.

키 배치는 레인 수별 프리셋으로 관리되어야 하므로, 4레인과 7레인은 서로 다른 키 배치를 가질 수 있습니다.

## 5. 동시 입력

빠른 편집 모드는 동시치기를 지원해야 합니다.

예를 들어 4레인에서:

- `D`와 `K`를 동시에 누르면 1번 레인과 4번 레인에 같은 스냅 시점으로 노트 배치 신호가 들어가야 합니다.

재생 중에는 타이밍을 맞추기 어렵기 때문에, 동시 입력도 개별 노트가 아니라 **같은 스냅 시점의 코드(chord)** 로 처리되어야 합니다.

## 6. 롱노트 입력

현재 선택된 노트 타입이 롱노트 가능 타입이면:

- **키 다운** = 롱노트 시작
- **키 업** = 롱노트 끝

같은 레인 키는 눌렀을 때 생성한 해당 롱노트의 소유권을 유지해야 하며, 여러 레인을 동시에 누르는 롱노트 입력도 지원해야 합니다.

또한 길이가 0인 롱노트는 기존 에디터의 롱노트 정리 규칙에 따라 일반 노트로 접히는 동작을 유지해야 합니다.

## 7. 빠른 편집 모드에서의 마우스 동작

빠른 편집 모드에서는 타임라인 마우스 동작이 바뀝니다.

- 좌클릭: 노트 삽입이 아니라 **구간 선택**
- 우클릭: **붙여넣기 기준 마커** 배치

### 구간 선택 흐름

1. 첫 번째 좌클릭으로 구간 시작점 또는 한쪽 경계를 잡습니다.
2. 두 번째 좌클릭으로 구간 끝점을 잡아 선택을 완성합니다.
3. 그 뒤 `Ctrl+C`로 선택된 구간의 노트를 복사할 수 있습니다.

### 붙여넣기 흐름

1. 우클릭으로 붙여넣기 기준 마커를 놓습니다.
2. `Ctrl+V`를 누르면 복사한 노트가 해당 마커를 시작점으로 하여 붙여넣어집니다.

## 8. 복사 / 붙여넣기 규칙

빠른 편집 모드에서:

- `Ctrl+C`는 선택 구간 안의 노트를 복사합니다.
- `Ctrl+V`는 붙여넣기 마커를 기준으로 노트를 붙여넣습니다.
- 붙여넣기된 노트도 기존 배치 규칙을 따라야 합니다.
- 중복 또는 충돌이 나는 노트는 삽입되면 안 됩니다.

가능한 한 원본의 상대 시간 차이와 레인 구성을 유지하되, 최종 반영은 항상 현재 차트의 유효성 규칙을 우선해야 합니다.

## 9. 수동 테스트 체크리스트

### 모드 전환

1. 빠른 편집 모드를 켭니다.
2. UI에서 모드가 켜졌음을 확인합니다.
3. 다시 끕니다.
4. 일반 타임라인 클릭 삽입이 다시 동작하는지 확인합니다.

### 재생 중 탭 노트 입력

1. 차트와 곡을 불러옵니다.
2. 탭 노트 타입을 선택합니다.
3. 재생을 시작합니다.
4. 레인 키를 박자에 맞춰 누릅니다.
5. 노트가 raw time이 아니라 snapped time으로 들어가는지 확인합니다.

### 동시치기

1. 빠른 편집 모드에서 재생을 시작합니다.
2. 두 개 이상의 레인 키를 동시에 누릅니다.
3. 여러 레인에 같은 스냅 시점으로 노트가 들어가는지 확인합니다.

### 롱노트

1. 롱노트 가능 타입을 선택합니다.
2. 재생을 시작합니다.
3. 레인 키를 누른 채 유지했다가 나중에 뗍니다.
4. 기대한 시작 / 끝 시간으로 롱노트가 생성되는지 확인합니다.
5. 여러 레인 동시 홀드도 확인합니다.

### 선택 + 복사 / 붙여넣기

1. 빠른 편집 모드를 켭니다.
2. 좌클릭 두 번으로 구간을 선택합니다.
3. `Ctrl+C`를 누릅니다.
4. 우클릭으로 붙여넣기 마커를 놓습니다.
5. `Ctrl+V`를 누릅니다.
6. 복사한 노트가 마커 기준으로 붙는지 확인합니다.
7. 중복 노트는 그대로 거부되는지 확인합니다.

## 10. 설계 제약

빠른 편집 모드는 새 에디터의 기존 아키텍처를 따라야 합니다.

- 핵심 배치 규칙은 UI 밖의 core에 둡니다.
- 위젯이 차트 데이터를 직접 바꾸지 않고 session 메서드를 호출합니다.
- 빠른 편집 입력도 기존 클릭 배치와 같은 스냅 / 중복 방지 규칙을 재사용합니다.
